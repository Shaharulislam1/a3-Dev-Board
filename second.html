Question-1: What are the different ways to select an element in the DOM?

    Answer:

  • 1. get by id - select an element using unique id.
  • 2. get by class name - select element using a shared class name.
  • 3. get by tag name - select element based on HTML tag.
  • 4. get by selector - select the element using a CSS - style
  • 5. get by selectorAll - select all element using a CSS- style

Question-2: What is the difference between innerHTML, innerText, and textContent ?

    Answer:

  1. 1. innerHTML- get or sets the HTML content inside an element.
  2. 2. innerText- get or sets the visible text inside an element.
  3. 3. textContent- get or sets the content inside an element , include hidden text.

Question-3: What is event delegation in the DOM?

Answer:

Event delegation is a technique in JavaScript, when keeping a single event listener for multiple elements so that it still works even when new element are added.

Question-4: What is event bubbling in the DOM?

Answer:

Event bubbling is a process where when an event occurs on an element, it propagates step by step to its parent and higher element. That is the event is first executed on the element where it occurred then its parent, than its parent and so on, continuously reaching the top of the Dom tree.
